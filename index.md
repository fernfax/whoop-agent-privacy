# Privacy Policy — OpenClaw Personal Agent

**Last updated:** 17 April 2026

## Overview

This application ("OpenClaw WHOOP Integration") is operated for personal use by a single user — the developer — and is not offered as a public service. It accesses third-party APIs (including WHOOP) solely to deliver personalized summaries, reminders, and notifications to the developer via a private Telegram channel.

## Data Collected

The application accesses data from the authenticated user's WHOOP account, including recovery scores, sleep, strain/cycle, and workout data. No data is collected from any other user.

## Data Usage

Data is used exclusively to generate personal briefings, reminders, and summaries delivered to the developer's own devices. Data is not used for analytics, advertising, machine learning training, or any commercial purpose.

## Data Sharing

No data is shared with third parties. No data is sold, transferred, or disclosed to any external service, advertiser, or partner.

## Data Storage

Data is stored on a private server under the developer's sole control and is accessible only to the developer. Data may be deleted at any time, and OAuth authorization can be revoked at any time through the WHOOP account settings.
